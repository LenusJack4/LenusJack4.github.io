About Me

Thank you for taking the time to view my portfolio.

Lost Soul

Play as a soul in search of something that he can't quite remember while he slowly loses his light. And find it before the darkness steals it all. Lost Soul was created during the 2021 Global Game Jame where the theme was "lost." Due to the game jam being in a new format this year to adapt to covid precautions, the games were not judged and were made purely for the experience. For this game I scripted a lot of the events, programmed some movement and health interactions, and fully animated the player character and enemy NPCs.

SER

SER is a selfish, egotistical robot who must go on a journey to repair himself both physically and emotionally. SER was created during the 2020 Global Game Jam where the theme was "repair" and SER won the award for "Best Use of Theme" at the LSU site. This is the first game I created with a team, and I was the sole gameplay/movement programmer for the project. The gameplay was built from scratch and without assistance from tutorials.

Galaxy Defender

This game is based off of the classic space shooter games. The player must navigate the ship through enemy fire and shoot enemy ships to rack up score. I made this game with the help of a few tutorials to nail the core gameplay loop as I familiarized myself with Unity programming, but implemented certain features on my own without help, such as the collectibles and the win condition. The real fun starts after gaining a score of 20,000.

Super Creative Racecar Game

This is the first game that I made entirely by myself and from scratch. This game is a 2D endless runner that procedurally generates as the player continues. As long as the player does not run out of health, they can play forever and keep increasing their score by collecting coins.

Block Breaker

This is the first game I ever completed. It is a simple game where the player must control a paddle to bounce a ball into blocks to break them, based off of the classic Atari Breakout. It was made with the assistance of tutorials, but I did not like the way the paddle felt to control so I was able to adjust the code to tune the game and paddle behavior to make it feel better more responsive and forgiving to players with slower reflexes without slowing down the game speed itself.
